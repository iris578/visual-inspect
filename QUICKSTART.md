# Quick Start Guide

Get VisBug running in 2 minutes!

## Step 1: Generate Icons (30 seconds)

1. Open `icons/generate-icons.html` in your browser
2. Click all 4 download buttons
3. Save the files in the `icons/` folder:
   - icon16.png
   - icon32.png
   - icon48.png
   - icon128.png

## Step 2: Load Extension (30 seconds)

### Chrome / Edge / Brave / Arc

1. Go to `chrome://extensions/`
2. Toggle on "Developer mode" (top-right)
3. Click "Load unpacked"
4. Select this `devtools-extension` folder
